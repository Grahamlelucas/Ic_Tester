Getting Started with IC Tester Pro

Welcome! This application tests 74-series ICs using an Arduino Mega 2560.

Step 1: Connect Your Arduino
  1. Plug Arduino Mega into USB
  2. Upload the IC Tester sketch via Arduino IDE
  3. Close Arduino IDE (it blocks the port)
  4. Click Scan then Connect
Step 2: Configure Pin Mapping
  1. Select chip from dropdown
  2. Enter Arduino pin for each chip pin in the Pin Mapping tab
  3. VCC/GND are auto-marked as PWR
  4. Click Validate, then optionally Save
Step 3: Run the Test
  1. Click Run Test
  2. Green = passed, Red = failed
  3. Check Output Log and Dashboard for details
Learning About 74-Series Chips
Chip Families
  • Logic Gates (7400, 7404, 7408, 7432)
  • Flip-Flops (7474, 7475)
  • Counters (7490, 7493)
  • Decoders (74138, 74139)
  • Shift Registers (74595)
Recommended Learning Path
  1. 7404 Inverter — simplest chip
  2. 7400 NAND Gate — universal gate
  3. 7408 AND, 7432 OR
  4. 7474 D Flip-Flop
  5. 7490/7493 Counters
Key Concepts
  • Truth Tables — all input/output combos
  • Edge Triggering — responds to transitions
  • Active Low — signal 'on' when LOW
  • Floating Inputs — unconnected = problems
Adding New Chip Definitions

Chip definitions are JSON files in the chips/ folder.

Required Fields
  • chipId — part number
  • name — human name
  • pinout — inputs, outputs, VCC, GND
  • arduinoMapping — default pin mapping
  • testSequence — the tests to run
{

  "name": "Quad 2-Input NAND",

    "inputs": [{"pin": 1, "name": "1A"}],
    "outputs": [{"pin": 3, "name": "1Y"}]
  },
  "testSequence": {"tests": [...]}

Troubleshooting
Arduino Not Found
  • Check USB cable
  • Close Arduino IDE
  • Click Scan to refresh
All Tests Failing
  • Double-check wiring
  • Verify VCC (5V) and GND
  • Check chip orientation
  • Try a known-good chip
Inconsistent Results
  • Check for loose wires
  • Add decoupling capacitors
  • Avoid long wire runs
